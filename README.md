# Mars-Rover-code

Code used for the 1/4 Scale Mars rovers at Lake Tuggeranong College.

## Folders

/arduino-mega-code contains the code for the arduino mega\
/esp32-code contains the code for the arduino mega\
/pi-code contains the code for the raspberry pi (Serial communication)

/PCB contains the PCB files to make it easier to get the pinout

/docs contains the code for the website (we could use it as a hub to easily go to any page on every rover)
